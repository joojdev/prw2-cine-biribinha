# 🎬 Cine Biribinha

Trabalho desenvolvido para a disciplina de **Programação para Web 2** do Instituto Federal de São Paulo — Campus São Carlos.

A aplicação consiste em uma página web que consome uma API CRUD hospedada no [MockAPI](https://mockapi.io).

---

## 🛠️ Tecnologias

- ⚛️ React + Vite
- 🔀 React Router DOM
- 🌐 Axios
- 🧩 Zod
- 🏗️ Arquitetura em camadas

## 🚀 Comandos

```bash
# Cria um .env baseado no exemplo do projeto
cp .env.example .env

# Edita o .env colocando a URL do seu backend de escolha
nano .env

# Inicia o servidor de desenvolvimento
npm run dev

# Gera o build de produção
npm run build

# Visualiza o build localmente
npm run preview
```
